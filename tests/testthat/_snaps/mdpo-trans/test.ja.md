## 図表

Rコードを書くことも出来ます：

```{r}
#| label: pyramid
#| fig.alt: ピラミッドの錯覚
#| fig.cap: 毎朝お日様が登ります
pie(
  c(Sky = 78, "ピラミッドの明るい方" = 17, "ピラミッドの影に入っている方" = 5), 
  init.angle = 315, 
  col = c("deepskyblue", "yellow", "yellow3"), 
  border = FALSE
)
```

::::::::::::::::::::::::::::::::::::: keypoints 


  - エピソードの内容が変わらない場合は`.md`ファイルを使いましょう
- コードによってエピソードの出力を作りたい場合は`.Rmd`ファイルを使いましょう
